# GitHub Copilot Setup Guide

This guide explains how to get authorized to use GitHub Copilot for the Gravity Dream project.

## What is GitHub Copilot?

GitHub Copilot is an AI pair programmer that helps you write code faster. It suggests whole lines or entire functions as you type.

## How to Get Access to GitHub Copilot

### Option 1: Free Access for Students

If you're a student, you can get GitHub Copilot for free through the GitHub Student Developer Pack:

1. Visit [GitHub Education](https://education.github.com/pack)
2. Click "Sign up for Student Developer Pack"
3. Verify your student status with:
   - A school-issued email address, OR
   - Documents proving enrollment (student ID, transcript, etc.)
4. Once verified, GitHub Copilot will be automatically enabled for your account

### Option 2: Free Access for Open Source Maintainers

If you're a verified open-source maintainer of popular projects, you may qualify for free Copilot access:

1. Visit [GitHub Copilot for Open Source](https://github.com/features/copilot)
2. Check if you meet the eligibility criteria:
   - Maintain a popular open-source project
   - Have significant contributions to open-source
3. Apply through GitHub's verification process

### Option 3: Individual Subscription

If you don't qualify for free access, you can subscribe to GitHub Copilot:

1. Go to [GitHub Copilot Settings](https://github.com/settings/copilot)
2. Click "Get access to GitHub Copilot"
3. Choose between:
   - **Copilot Individual**: ~$10/month or $100/year
   - **Copilot Business**: For organizations, ~$19/user/month
4. Complete the payment process

## How to Enable GitHub Copilot

Once you have access:

### For Visual Studio Code:

1. Install the [GitHub Copilot extension](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot)
2. Sign in with your GitHub account when prompted
3. Authorize the extension
4. Start coding - Copilot suggestions will appear automatically

### For JetBrains IDEs (Rider, IntelliJ, etc.):

1. Go to Settings/Preferences → Plugins
2. Search for "GitHub Copilot"
3. Install the plugin
4. Restart your IDE
5. Sign in with your GitHub account
6. Start coding - Copilot suggestions will appear automatically

### For Visual Studio:

1. Go to Extensions → Manage Extensions
2. Search for "GitHub Copilot"
3. Install the extension
4. Restart Visual Studio
5. Sign in with your GitHub account
6. Start coding - Copilot suggestions will appear automatically

## Verifying Your Access

To verify that GitHub Copilot is enabled:

1. Visit [GitHub Copilot Settings](https://github.com/settings/copilot)
2. You should see your subscription status
3. If you see "You don't have access to GitHub Copilot", you need to:
   - Apply for student access, OR
   - Purchase a subscription, OR
   - Check if your organization provides access

## Troubleshooting

### "Not authorized" Error

If you see a "not authorized" error:

1. **Check your subscription**: Visit [GitHub Copilot Settings](https://github.com/settings/copilot)
2. **Verify IDE extension**: Make sure the Copilot extension/plugin is properly installed
3. **Sign in again**: Try signing out and signing back in to your GitHub account in your IDE
4. **Check organization policies**: If you're part of an organization, check if they have Copilot policies
5. **Network issues**: Ensure you're not behind a firewall blocking GitHub services

### Still Having Issues?

- Check [GitHub Copilot Documentation](https://docs.github.com/en/copilot)
- Visit [GitHub Copilot FAQ](https://github.com/features/copilot/faq)
- Contact GitHub Support if you believe you should have access but don't

## For Organization Owners

If you're managing an organization and want to enable Copilot for your team:

1. Go to your organization settings
2. Navigate to "Copilot" under "Code, planning, and automation"
3. Enable Copilot for your organization
4. Set policies for who can use it
5. Users will then have access when working on organization repositories
