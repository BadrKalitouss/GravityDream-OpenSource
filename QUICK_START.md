# Quick Start: Fixing "Not Authorized" for GitHub Copilot

If you're seeing a "not authorized" error when trying to use GitHub Copilot, follow these quick steps:

## Step 1: Check Your Access (1 minute)

Visit: https://github.com/settings/copilot

**What you'll see:**
- ✅ "You have access to GitHub Copilot" → You have access, skip to Step 3
- ❌ "You don't have access" → Continue to Step 2

## Step 2: Get Access (Varies)

### Are you a student? → Get FREE access
1. Go to: https://education.github.com/pack
2. Sign up with school email or upload enrollment proof
3. Wait 1-3 days for approval
4. ✅ Access will be automatically granted

### Contributing to open source? → Get FREE access
1. Apply at: https://github.com/features/copilot
2. Must maintain popular open-source projects
3. GitHub will review your application

### Everyone else → Subscribe
1. Go to: https://github.com/settings/copilot
2. Click "Get access to GitHub Copilot"
3. Choose a subscription plan (see [GitHub's pricing](https://github.com/features/copilot#pricing))
4. ✅ Immediate access

## Step 3: Install Extension (2 minutes)

### VS Code (Most Common)
1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search "GitHub Copilot"
4. Click Install
5. Sign in when prompted

### JetBrains (Rider, IntelliJ, etc.)
1. Open your IDE
2. Settings → Plugins
3. Search "GitHub Copilot"
4. Install and restart
5. Sign in when prompted

### Visual Studio
1. Extensions → Manage Extensions
2. Search "GitHub Copilot"
3. Install and restart
4. Sign in when prompted

## Step 4: Test It (30 seconds)

1. Create a new file (e.g., `test.cs`)
2. Type: `// Function to add two numbers`
3. Press Enter
4. ✅ You should see gray "ghost text" suggestions

**Still not working?**
→ Restart your IDE
→ Sign out and sign back in
→ See [COPILOT_SETUP.md](COPILOT_SETUP.md) for detailed troubleshooting
